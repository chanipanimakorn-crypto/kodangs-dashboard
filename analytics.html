🔍 Analytics Framework

การอ่านตัวเลขแบบ CEO

Dashboard นี้ไม่ใช่แค่ report — เป็น decision-making framework. แต่ละ metric ถูกคัดมาเพื่อตอบคำถาม CEO 3 ข้อหลัก:

1. Velocity — กำลังจะไปทันเป้าไหม?

  • Pace projection = MTD ÷ วันที่ผ่านไป × 30 → บอกแนวโน้มก่อนสิ้นเดือน 7 วัน
  • Peak comparison (vs ม.ค.) → ถ้า pace ต่ำกว่า peak เกิน 15% = warning sign
  • Target Gap → จำนวนเงินที่ต้อง bridge ทุกเดือน ไม่ใช่ตัวเลข vanity

2. Quality — กำไรจริงหรือเปล่า?

  • Gross margin % — ถ้ายอดขึ้นแต่ margin ตก = discount เยอะไป / ต้นทุนขึ้น
  • ต้นทุนสต็อก vs มูลค่าสต็อก — ปัจจุบัน ฿1.82M cost vs ฿8.56M value → potential GP ฿6.74M ล็อกอยู่ในคลัง

3. Execution — ทีม block อะไร?

  • Stuck tasks ของ Noon — tasks ที่ CEO ต้อง unblock ด้วยตัวเอง (ไม่ใช่ assign ต่อ)
  • Team overdue — pattern ของ tasks ที่ค้าง = ชี้ระบบที่พัง, ไม่ใช่คน

♟️ CEO Strategy · สิ่งที่ Noon ควรโฟกัส

ไม่ใช่ task list — แต่เป็นสิ่งที่ต้องคิดและตัดสินใจเอง ใครทำแทนไม่ได้

1. ไล่ของค้างสต๊อก ฿8.56M — เปลี่ยนสินทรัพย์ให้เป็นเงิน

มี task "QA: สงสัยว่ามีสินค้าค้างสต๊อก" (urgent) ค้างอยู่ — ไม่ใช่ปัญหาเล็ก. Stock 11.5 เดือน = cash flow ติด.

สิ่งที่ต้องทำ: รัน quarterly dead stock analysis (Q2 ควรเริ่มแล้ว) · ให้ Kanjana + Front pull list 90-day no-sale · ตัดสินใจ flash sale หรือ wholesale clearance · เป้า: ปล่อย stock ฿1-2M ใน 60 วัน

2. ปลดล็อก Kodangs US — Invoice + Eye contract + LOGO register

มี 3 tasks เกี่ยวกับ US expansion ค้างอยู่ในโต๊ะ Noon (Invoice stuck, Eye urgent, LOGO to-do). ทุกอย่าง block การเปิดตลาด US.

สิ่งที่ต้องทำ: block 2 ชม. วันนี้ clear 3 งานนี้ให้จบ · ถ้าไม่มี lawyer ที่ US ยังไม่คอนเฟิร์ม = ต้องหาใหม่ · ถาม Panpornpach ว่า shipping 1st round ติดที่ไหนจริงๆ

3. เปลี่ยนจาก "push ads" → "ขยายช่องทาง" เพื่อไปเป้า ฿4M

Pace ปัจจุบัน ฿973K/เดือน — แค่ boost Facebook ads ไม่พาไป ฿4M. ต้องเพิ่ม structural leverage:

สิ่งที่ต้องตัดสินใจ: (a) flagship Song Wat จะเปิดเมื่อไหร่ + projection เท่าไหร่? (b) Live selling มี SOP ยัง? ใครขายได้ scale? (c) wholesale US — Iris Malaysia ready order ครั้งที่ 2 หรือยัง?

4. ดูแล Margin ที่ 62.1% — ไม่ให้ถูก Shopee/TikTok fee กัด

ข่าววันนี้: Shopee เพิ่ม 5% technical support fee + TikTok Shop ขึ้น commission. Margin ช่องทางนี้จะหดภายใน 3 เดือน.

สิ่งที่ต้องทำ: ดึง data channel split (Shopee/TikTok vs IG/LINE/หน้าร้าน) · คำนวณว่าถ้าขึ้น fee อีก 5% จะกิน margin เท่าไหร่ · วางแผนดัน traffic ไปช่อง direct (LINE, website) ที่ margin สูงกว่า

5. INTERN 2026 — build ระบบคนจูเนียร์ที่ scale ได้

Task "PROJECT : INTERN 2026" in progress — ถ้าวาง framework ดี = ได้ผลงานถูก 50% · content + ops · 1 ปีข้างหน้า

CEO ต้อง decide: จำนวน intern, role (content / ads / ops), mentor ใคร, KPI, เริ่ม batch เมื่อไหร่ · Kanjana กับ Tantai รับลูกต่อได้
